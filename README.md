# PKSFZ

Pemulih kata sandi file ZIP yang ditenagai oleh Fcrackzip, John The Ripper dan Hashcat.

## Instalasi

```
sudo apt-get update
sudo apt-get install git
git clone https://github.com/fixploit03/PKSFZ.git
cd PKSFZ
chmod +x *.sh
sudo ./run.sh
```
